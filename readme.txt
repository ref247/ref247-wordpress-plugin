== Description ==

Ref247 Affiliate Tracking is a comprehensive WordPress plugin that integrates with the Ref247.io affiliate management platform. It enables site owners to build and manage an affiliate program directly from their WordPress dashboard.

**Key Features:**

* **Real-time Affiliate Dashboard** – View organization statistics, commissions, and affiliate trends at a glance
* **Commission Tracking** – Automatic commission calculation and management for referrals
* **WooCommerce Integration** – Seamlessly track e-commerce purchases as referral actions
* **Multi-Form Support** – Integrate with WPForms, Gravity Forms, Contact Form 7, and more
* **Affiliate Signup Page** – Auto-generated signup page for affiliate registration
* **Commission Rules** – Flexible commission structures managed in the Ref247.io dashboard
* **Analytics & Charts** – Visual insights into commission trends and referral performance
* **Multi-Currency Support** – Track commissions across different currencies
* **User Role Distribution** – Monitor team composition and user statistics

**Affiliate Program Management:**

With Ref247, you can:
- Create and manage multiple affiliate campaigns
- Set up flexible commission structures and rules
- Track affiliate performance in real-time
- Manage referral links and marketing assets
- Monitor pending and paid commissions by currency
- View affiliate statistics and trends

**Easy Integration:**

Simply add your API credentials in the plugin settings, and the plugin automatically:
- Validates your organization access
- Loads your campaigns and commission structures
- Creates an affiliate signup page
- Integrates with WooCommerce to track purchases
- Sets up form integration for lead tracking

**Perfect For:**
- E-commerce stores using WooCommerce
- Marketing agencies managing affiliate programs
- SaaS companies tracking referral signups
- Content creators with partner networks
- Any business wanting to scale through affiliate marketing

For more information and to get started, visit [Ref247.io](https://ref247.io)

== Installation ==

1. **Automatic Installation:**
   - Go to Plugins > Add New in your WordPress admin
   - Search for "Ref247 Affiliate Tracking"
   - Click "Install Now" and then "Activate"

2. **Manual Installation:**
   - Download the plugin from WordPress.org
   - Extract the zip file to `/wp-content/plugins/` directory
   - Go to Plugins menu and activate "Ref247 Affiliate Tracking"

3. **Initial Setup:**
   - Go to the Ref247 menu in your WordPress admin
   - Visit the Settings tab
   - Enter your API Key and API Secret from [Ref247.io](https://ref247.io)
   - Click "Save API Credentials"
   - Click "Force Refresh" to load your organization data
   - Visit the Dashboard tab to view your affiliate metrics

== Frequently Asked Questions ==

= What is Ref247? =

Ref247 is a complete affiliate tracking and management platform. This plugin connects your WordPress site to Ref247.io, enabling you to manage your entire affiliate program from your WordPress dashboard.

= Do I need a Ref247.io account? =

Yes, you need an active Ref247.io account to use this plugin. You can sign up for free at [Ref247.io](https://ref247.io).

= How do I get my API credentials? =

Log in to your Ref247.io account, navigate to your organization settings, and generate API credentials. Copy the API Key and API Secret into the plugin settings.

= Is WooCommerce required? =

No, WooCommerce is optional. The plugin works standalone for tracking affiliate signups and referrals via forms. WooCommerce integration is available if you're selling products.

= Which form builders are supported? =

The plugin supports:
- WPForms
- Gravity Forms
- Contact Form 7
Additional integrations can be configured through Ref247.io.

= Can I use this with multiple currencies? =

Yes, Ref247 supports unlimited currencies. You can set up commissions in any currency and track them separately.

= How often is the dashboard data updated? =

Dashboard data is fetched from Ref247.io in real-time when you visit the page. Commission data reflects the current state of your affiliate program.

= Is my data secure? =

Yes. The plugin communicates with Ref247.io over HTTPS using API credentials. All data transmission is encrypted, and your credentials are stored securely in WordPress options.

= Can I manage multiple organizations? =

Currently, the plugin works with one organization per WordPress installation. You can manage multiple campaigns and affiliates within that organization.

= What permissions does the current user need? =

Only users with the "manage_options" capability (typically administrators) can access the Ref247 settings and dashboard.

= Is there support available? =

Yes! Visit the support page [support.ref247.io](https://support.ref247.io) for assistance.
